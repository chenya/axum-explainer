Middleware

Middleware adds behaviour that runs around every request — logging, auth, timeouts, CORS. Each .layer() call wraps the entire router in a shell. The last layer you write becomes the outermost shell and runs first.

Nested shells

Each .layer() wraps everything above it. A request enters the outermost shell first.

TraceLayer — logs every request and response
TimeoutLayer — cancels slow requests after 10s
CorsLayer — adds CORS headers to responses
🔀 Your Router + Handlers — the actual logic
⚠️

Warning

The order rule: Layers are written bottom-up in code but execute top-down on the request. The last .layer() you call wraps everything — so it runs first on every incoming request and last on every outgoing response.

Auth middleware

Write your own middleware with axum::middleware::from_fn. An auth layer validates the session cookie and either calls next.run(req) to continue or returns a 401 immediately.

📊

TraceLayer

TraceLayer::new_for_http() integrates with the tracing crate. Every request gets a span with method, path, and status. Set RUST_LOG to see structured logs.
